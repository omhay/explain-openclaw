### Daemon LaunchAgent Stability ([`4ebefe647`](https://github.com/openclaw/openclaw/commit/4ebefe647), [`b97571142`](https://github.com/openclaw/openclaw/commit/b97571142))

Two commits improve LaunchAgent lifecycle management:

1. **KeepAlive simplification** — Reverts crash-only KeepAlive policy to simple `KeepAlive=true`, preventing unintended service termination while preserving restart hardening.

2. **Restart sequencing** — Implements robust restart with PID wait (10s timeout), proper `bootout → wait → bootstrap → kickstart` sequence, and proxy environment variable forwarding (HTTP_PROXY, HTTPS_PROXY, NO_PROXY, etc.) via new `service-env.ts` module.

### Account-Scoped Pairing Isolation ([`91a3f0a3f`](https://github.com/openclaw/openclaw/commit/91a3f0a3f))

Enforces strict account isolation for pairing allowlists and pending requests:

- `readNonDefaultAccountAllowFrom()` and sync variant read from account-scoped paths
- `shouldIncludeLegacyAllowFromEntries()` prevents fallback to legacy unscoped allowlist
- Upsert operations use normalized account ID for matching

**Audit 2 Claim 5 relation:** Strengthens account isolation, preventing cross-account pairing access that could enable self-approving agent scenarios.

### Agent Routing Bindings ([`96c770252`](https://github.com/openclaw/openclaw/commit/96c770252))

Adds account-scoped routing bindings:

- New CLI commands: `agents bindings`, `agents bind`, `agents unbind`
- Explicit agent-to-channel account binding storage
- `resolveBindingAccountId()` interface for plugin-specific account resolution

**Audit 2 Claim 5 relation:** Prevents cross-channel account access by enforcing explicit binding relationships.

### Android Device Auth Requirement ([`a87d961eb`](https://github.com/openclaw/openclaw/commit/a87d961eb))

Makes device auth store mandatory for Android gateway sessions:

- `DeviceAuthTokenStore` interface with mandatory `saveToken()` method
- Gateway session constructor requires device auth store parameter
- Ensures token persistence and prevents unauthenticated device connections
